What is this all about?

This is a cellular automation zero-player game invented by mathamatician John Conway in 1970 called Conway's game of life.

The universe of this game is an infinite 2D grid of cells that can be either alive or dead. On each turn, the neighbours of a given cell determine if the cell is alive or dead based on a series of rules. For a more in-depth explanation, check out this video or the LifeWiki page.

Try clicking on some cells on the editor and then hit the play button to see how they interact together.
