Purpose

The purpose of this site is to allow users to explore the phenomenon of cellular automaton within the bounds of the game of life. The user can experiment with their own creations, view over 1000 previously created templates, and apply different themes to create beautiful life forms.

Tabs

Here are some details about each of the tabs on the site.

  • Board The game of life game board.
  • Templates A massive list of game of life blueprints. Over 1000! Don't forget to mouse over (or press for mobile) to preview the template.
  • Themes Themes gives you the ability to change how the game of life board looks. Be sure to turn on generations for some neat effects, but for larger templates, it is best to turn this feature off.
  • About General details about the site, the game of life, and some interesting links.

Development

I originally started on this site because I wanted to learn how to use the javascript framework Angular 4. It was never meant to be a fully operational website. Unfortunately, I broke my ankle playing soccer which gave me a lot more free time. Lacking inspiration I kept pushing this site further and further until it looks like the site you see today.

The game of life algorithm I used is called Tony Finch's List Life algorithm. I also used this site as inspiration for my choice of algorithm and used the site to help me through learning the best way to utilize the html canvas element.

For future development I would like the site to use the Hashlife algorithm since the list life algorithm doesn't handle large cellular automaton. I would like to spend some time refining a few features. That's about it really. Site is pretty much done. I have no interest in adding a backend on this project.
